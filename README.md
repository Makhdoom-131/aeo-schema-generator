# 🧠 AI Schema Generator

Instantly generate Google AI Overviews & ChatGPT optimized JSON-LD schemas from plain text. Turns simple Q&As into structured FAQ and Article schemas.

## ✨ Features

- **FAQ Schema Generator:** Automatically parses Q&A text into valid JSON-LD.
- **Article Schema:** Adds Author and Publish Date metadata.
- **Answer Engine Optimization (AEO):** Formats data specifically for LLMs and AI search.
- **Zero Backend:** Pure frontend string parsing. No API keys needed.
- **Clean Architecture:** HTML, CSS, and JS completely separated.

## 🛠 Tech Stack

- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Logic:** Regex parsing & JSON object structuring
